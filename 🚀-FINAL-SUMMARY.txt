🚀 HOW TO DEPLOY (3 METHODS)
================================================================================

┌─────────────────────────────────────────────────────────────────────────────┐
│ METHOD 1: WRANGLER CLI (FASTEST - 2 MINUTES)                                │
└─────────────────────────────────────────────────────────────────────────────┘

cd video-downloader

# Step 1: Login to Cloudflare
wrangler login

# Step 2: Deploy
wrangler pages deploy public --project-name video-downloader

✨ DONE! Your site will be live at: https://video-downloader.pages.dev


┌─────────────────────────────────────────────────────────────────────────────┐
│ METHOD 2: GITHUB + CLOUDFLARE PAGES (AUTOMATED CI/CD)                       │
└─────────────────────────────────────────────────────────────────────────────┘

# Step 1: Push to GitHub
cd video-downloader
git remote add origin https://github.com/YOUR-USERNAME/video-downloader.git
git branch -M main
git push -u origin main

# Step 2: Connect to Cloudflare Pages
1. Go to: https://dash.cloudflare.com
2. Click: Pages → Create a project → Connect to Git
3. Select: video-downloader repository
4. Configure:
   - Project name: video-downloader
   - Production branch: main
   - Build command: (leave empty)
   - Build output directory: public
5. Click: Save and Deploy

✨ DONE! Auto-deploys on every git push!


┌─────────────────────────────────────────────────────────────────────────────┐
│ METHOD 3: DIRECT UPLOAD (NO GIT REQUIRED)                                   │
└─────────────────────────────────────────────────────────────────────────────┘

1. Go to: https://dash.cloudflare.com
2. Click: Pages → Create a project → Upload assets
3. Drag the 'public' folder
4. Name: video-downloader
5. Click: Deploy site

✨ DONE! Live immediately!
